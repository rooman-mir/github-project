# Calculator Project - Git & GitHub Capstone Project

## About
A beginner-friendly Python calculator project developed to demonstrate Git & GitHub skills including branching, merging, and conflict resolution.

## Features (5 Functions)
1. Addition
2. Subtraction
3. Multiplication
4. Division (with error handling)
5. Modulus (with error handling)

## Branches
- **main** - Initial project with basic calculator
- **developer1** - Added if-else error handling for division and modulus
- **developer2** - Added try-except error handling for division and modulus
